# Picoin Layer 2

This directory contains future Layer 2 work for Picoin.

Current modules:

- `picoin-forge-ai`: MVP for Picoin Forge AI, a repo analysis and Linux deployment agent designed as a future verifiable compute worker for Picoin.
- `picoin-forge-l2`: MVP base for Proof of Available Compute, where workers earn simulated PI by verified compute capacity per epoch.

Layer 2 goals:

- Coordinate useful off-chain compute jobs without making Layer 1 nodes run heavy workloads.
- Use Picoin staking for access, reputation, and provider accountability.
- Reward compute providers from Picoin reserve and task settlement flows.
- Keep Layer 1 focused on consensus, payments, staking, validation, and final settlement.
